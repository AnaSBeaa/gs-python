# Global Solution 2026 - Transmissão de Energia Solar Espacial

## Sobre o Projeto

Este projeto foi desenvolvido para a Global Solution com o objetivo de explorar uma possível aplicação da Indústria Espacial para atender à crescente demanda mundial por energia.

A proposta consiste em simular um sistema de transmissão de energia solar espacial, no qual satélites captam energia solar em órbita e a transmitem para a Terra por meio de feixes laser direcionados. O sistema desenvolvido permite realizar cálculos relacionados à eficiência da transmissão, perdas energéticas e estimativas de geração de energia, utilizando conceitos fundamentais de programação em Python.

Além da implementação computacional, o projeto também aborda aspectos de sustentabilidade, inovação tecnológica e aplicações futuras da energia solar espacial.

---

## Objetivos

- Simular conceitos relacionados à transmissão de energia sem fio.
- Demonstrar aplicações da Indústria Espacial para problemas terrestres.
- Aplicar conceitos de programação aprendidos no primeiro semestre.
- Desenvolver uma solução interativa baseada em menus.
- Realizar cálculos e análises sobre eficiência energética.

---

## Funcionalidades

O sistema possui um menu interativo contendo funcionalidades como:

- Descrição da solução proposta.
- Cálculo da eficiência da transmissão de energia.
- Estimativa de energia recebida na estação terrestre.
- Simulação de perdas energéticas durante a transmissão.
- Relatórios e análises dos resultados obtidos.
- Encerramento seguro do sistema.
